# ARCS Restaurant Management Website

## Features

- Fully responsive design
- English / Spanish translation system
- Persistent language settings with localStorage
- Dynamic announcement rendering
- Dynamic employee resource cards
- Scroll reveal animations
- Mobile-first navigation
- Netlify contact form integration

## Built With

- HTML5
- CSS3
- Vanilla JavaScript
- Font Awesome
- Netlify
